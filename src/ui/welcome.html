What AccessAgent Does

AccessAgent automatically repairs websites to make them more accessible. It runs on every page you visit. It fixes missing image descriptions, labels unlabeled buttons, repairs heading structure, dismisses cookie popups, detects CAPTCHAs, and more.

You don't need to do anything — it works automatically. But you also have keyboard shortcuts and a voice agent for extra control.

Keyboard Shortcuts

  • Control + Shift + A Toggle voice agent on and off
  • Control + Shift + S Hear a summary of the current page
  • Control + Shift + M What Am I Missing? accessibility gap report

On Windows or Linux, use Alt instead of Control. You can customize these shortcuts at any time by visiting chrome extensions shortcuts in your browser settings.

Voice Agent

Press Control + Shift + A to activate the voice agent. Then speak naturally. Here are some things you can say:

  • "Click" followed by a button or link name
  • "Go to" followed by a website address
  • "Scroll down" or "scroll up"
  • "Next heading", "next button", or "next link"
  • "What's on this page?" for a page summary
  • "What am I missing?" for an accessibility gap report
  • "Fill" a field name "with" a value
  • "Dismiss this popup" to close modals
  • "Go back" or "go forward"
  • "Stop" to stop speaking
  • "Help" to hear all available commands

How It Works

1. Tier 1 — Instant Repair. Runs in under 50 milliseconds on every page. Fixes images, buttons, form labels, headings, landmarks, and focus traps. No internet needed.
2. Tier 2 — Smart Repair. Automatically dismisses cookie consent banners. Detects CAPTCHAs and tells you if there's an audio alternative. Announces page changes on single-page apps.
3. Tier 3 — Vision AI. Optional. If you add an API key in settings, AccessAgent can capture a screenshot and describe what's visually on the page. This is the "What Am I Missing?" report.

Privacy

AccessAgent collects no data. No analytics. No tracking. If you add an API key for Tier 3, page content is sent to your chosen provider only. Your API key never leaves your device. The voice agent can run fully on-device with the Whisper option in settings.
